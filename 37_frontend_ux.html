3.7. Front-end UX

3. Nuestro rol en UX

Pregunta

1. ¿Cuál es el rol principal de un front-end en el desarrollo de un proyecto?

Respuestas

Diseñar la interfaz gráfica del producto.

Crear la estructura técnica del producto utilizando lenguajes de programación web.

Realizar investigaciones de mercado para identificar las necesidades de los usuarios.

Redactar textos llamativos para el producto digital.1

Pregunta

2. ¿Qué significa que el rol de front-end sea híbrido entre UX y programación?

Respuestas

El front-end debe tener habilidades tanto en diseño gráfico como en desarrollo web.

El front-end debe combinar tareas de investigación de usuario con tareas de programación.

El front-end debe desempeñar tareas de desarrollo y previas al desarrollo de un proyecto.

El front-end debe trabajar en estrecha colaboración con el equipo de marketing y publicidad.

Pregunta

3. ¿Cuál es uno de los puntos fuertes del rol de front-end?

Respuestas

Creatividad visual y diseño gráfico.

Habilidades de comunicación verbal y escrita.

Programación web y estructura técnica.

Análisis de datos y estadísticas.

Pregunta

4. ¿Por qué se considera una carga de trabajo extra el rol de front-end?

Respuestas

Porque implica diseñar interfaces gráficas complejas.

Porque requiere realizar investigaciones exhaustivas sobre las necesidades de los usuarios.

Porque combina tareas de programación con tareas previas al desarrollo del proyecto.

Porque implica gestionar la promoción y la comercialización del producto.
